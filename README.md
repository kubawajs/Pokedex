# Pokedex
Example .NET Core application which works like Pokedex from Pokemon anime.

## Usage

```
# swagger documentation
```

## Technologies used

* ASP .NET Core 2.2.0-preview3
* Blazor & Blazor Server 0.7.0
* Swagger & NSwag 12.0.0
